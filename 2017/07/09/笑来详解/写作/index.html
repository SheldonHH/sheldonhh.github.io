关于写作的思考和总结

Posted on 2017-07-09

“这个时代，普通人可以通过写书实现财富自由” — 笑来

1. 重复销售自己的时间

写作可以把自己的同一段时间重复销售，很多次很多次的极少数选择之一。

  1. 有效沟通是最重要的技能之一
  2. 一个人的沟通效率，从底层决定了一个人最终能够获得的资源以及质量

演讲和写作都是比较私下交流更有价值和更有效率的方式

中国能发表文字的远低于1%

新东方人数400到1200人一堂课

写书出版，九个月十几年的畅销书

大受欢迎的内容：一定是思考能力和分析能力出众的结果，包括故事和剧本。

2. “输出”倒逼”输入”

深入学习和思考，训练清晰逻辑

3. 出发点：

  1. 对人有用
  2. 做到最有用
  3. 对更多人有用
  4. 多长时间有用
  5. 如何让人优雅地回馈我

4. 记录帮助理解

只要你记笔记，就比不记笔记的同学要强

记录本身能够帮助我吗理解并熟悉内容

5. 教是最好的学

  1. 写作不用依赖他人可以独自完成教的过程

  2. 发表和的真实反馈很珍贵

  3. 给别人做榜样（孩子，爱人）

    ​

6. 写作不靠灵感

如弹琴，如篮球，这是训练的产物，（波动型上升，要熬过周期）

耐心真的很重要

7.记录痛苦，反思准备

尴尬的三点来源

  1. 高估自己（90%以上的司机坚信自己的驾驶水平在中等以上）

  2. 低估工作量 (Kobe work Ethics, Gary Player)

  3. 错误的方法 （解决方法： 用）

    ​

Gary Player

最后

只要真写就能真思考

时间可以让人和人之间产生天壤之别
