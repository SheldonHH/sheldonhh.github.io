scp

  • download the file: scp user@ip: server_path local_path

    • eg: scp -p22 ~/test.txt root@192.168.31.193:~/tmp/test1.txt // normally the port number is 22, but it is configurable

    • scp -p22 ~/test.txt root@192.168.31.193:~/tmp/test1.txt

      ​

  • download directory: scp -r user@ip: server_path local_path

    • eg: scp -r -p22 root@192.168. 31.193:~/tmp ~/ copy the directory to the server

  • Upload file: scp local_path user@ip:server_path
