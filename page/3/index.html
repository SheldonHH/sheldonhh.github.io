Hexo Create Tags / Hexo 如何创建新的Tags

Posted on 2016-09-23

Terminal 进入yourname.github.io folder之后

  1. 新建一个page，在/source/里面显示为一个新的文件夹tags

  1. 打开tags文件夹中的index.md

  1. 在菜单中添加链接。编辑 主题配置文件 ，添加 tags 到 menu 中，如下:

yourname.github.io/themes/next/_config.yml

另外一个同名文件位于yourname.github.io/_config.yml的

不要找错

/tags 是根据source里面的顺序排列

回顾： 为什么你总是半途而废1

Posted on 2016-09-23

今天笑来老师的文章可以总结为

不解释，调用元认知，可增长

执行力和其他能力一样，是一种可以通过后天锻炼的能力。

1. 不解释

坚持原则：

绝不个他人公开争吵

对于网上的负面言论，最好的方式就是不解释

回应一个高分男生的要求，提供价值（物质和情感）但不求回报

2. 调用元认知

当你面对非议或则不和离得指责时候，想发大火时候，可以启动一个弹窗模式，问自己发脾气能有助于解决问题吗

  1. 发脾气能有助于解决问题吗？
  2. 有没有必要为这件事情浪费自己的注意力呢？
  3. 同样的时间有没有更重要的事情可以做呢？

3. 可增长（growth mindset）

小时候的教育很多时候缺乏逻辑

毅力，耐心，勤奋，我们能做的也只是在精神上督促自己，或者是看到那些名人的故事，给自己打气

实际上，优秀品质绝非意念就能够养成的，而是靠科学的研究和合理的方法培养起来的

记住 EQ和IQ都是可以通过锻炼学会的
