Posted on 2016-09-23
  1. 你不需要检测，因为骨子里，就不愿意做的，需要坚持才行的事情，基本就做不好
  2. 不需要坚持就能做好的事情需要三个技巧
  • 一念一世界，在开始做事情前，先就该事情赋予重大意义。

写出想象的细节

  1. 如果没有这项技能，那么现在有什么事情我根本就做不了？根本没有机会？
  2. 在将来我会遇到什么样的困难？会失去什么可能的机会
  3. 如果我最终仍然没有掌握这项技能，我就和那些人是一样的？他们的生活究竟是多样的凄惨？
3. 通过社交来学习

“真好玩”，“停不下来”，“要能多玩儿一会儿就更好了” 的事情

Hexo Create Tags / Hexo 如何创建新的Tags

Posted on 2016-09-23

Terminal 进入yourname.github.io folder之后

  1. 新建一个page，在/source/里面显示为一个新的文件夹tags

  1. 打开tags文件夹中的index.md

  1. 在菜单中添加链接。编辑 主题配置文件 ，添加 tags 到 menu 中，如下:

yourname.github.io/themes/next/_config.yml

另外一个同名文件位于yourname.github.io/_config.yml的

不要找错

/tags 是根据source里面的顺序排列
